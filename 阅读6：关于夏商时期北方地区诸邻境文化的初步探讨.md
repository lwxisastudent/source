#### 河南、河北龙山为广义河南龙山

| 河南王湾型                                                   | 河北邯郸涧沟型                                       |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| 圆形袋穴；一般没有深窖                                       | 多直壁圆坑，不规则状锅底形坑；深窖水井               |
|                                                              | 细石器。盛行环状石斧（即大型石环——夏家店下层特色）   |
| 灰胎黑陶，温度高，无彩绘；点刺纹、方格纹                     | 红胎黑陶，烧成温度低，有极少彩绘；楔形点纹、绳切纹   |
| 鬲晚期消失，有夹砂粗红/黑陶缸                                | 多鬲、甗，无夹砂陶缸                                 |
| 鬶多作圆流或卷筒状流（盉）                                   | 晚期鬶多作尖咀流                                     |
| 几乎没有折腹盆                                               | 多折腹盆                                             |
| 鼎足剖面矩形，有的边缘有扉（夏文化早期相似——分布地域与夏文化相同：河南黄河以南、山西省西南） | 鼎足剖面扁三角或扁圆，有的面上有沟槽（先商文化相似） |

### 1.河北龙山涧沟型

- 分布：河北太行山东麓一线、山西太行山西麓一线，河南北部（先商漳河、辉卫分布区域）
- 遗址
  - 早期：涧沟遗址、磁县下潘汪下层
  - 中期：龟台寺
  - 晚期：汤阴白营上层，下潘汪上层（都有鬼脸式鼎足）
- 发现
  1. 涧沟遗址水井：汲水时坏而沉入井底的陶器；黄河流域最早水井
  2. 汤阴白营聚落：圆形房基横纵排列成行（有大小、精陋之分），**夯筑**，白灰涂墙，有的壁角敷有草拌泥，密集小柱子洞，圆形微凸灶（可能反应一夫一妻个体家庭，但排列整齐也可能是处在母权制“家长制家庭公社”的过渡期），有小孩作奠基材料（可能是蓄奴现象）
  3. 涧沟遗址丛葬坑（HJ2H18）：许多人被同时处死（血族复仇或战俘处理方式）
  4. 涧沟遗址剥人头皮：附近未发现肢骨的人头骨（砍下来）和浅切割痕
     - 民族志：猎取人头、剥人头皮，有的地方作胜利品，有的地方血祭使用
- 没有发现金属工具

#### 伯益作井

- 伯益/化益/相翳/大费
  - 参与了禹治水事业。文献：造井归给伯益而不是夏君主。夏早期遗址未发现井
  - 秦和赵的祖先
  - 玄鸟陨卵而生*（是吗？是女修屯玄鸟卵生大业，娶女华生大费）*商契也是玄鸟陨卵所生，地域相同
    - 卵生：东夷
    - 玄鸟：河北龙山*（大汶口鸟崇拜也是东夷，说到河北龙山有些附会，纯粹是为了附会所发现的水井）*
  - **涧沟早于伯益年代**
  - 益避禹之子于箕山之阴（一说河南阳城县南；还有多种说法）

### 2.河北龙山雪山型

- 分布：河北北部长城内外一带（**夏家店下层**分布区域的南部。——夏家店下层分布区域广，北边已超过辽宁西拉木伦河。两者发展直接相接）
- **细石器**比涧沟多
- 有鬲无斝；多折腹曲腹平地碗，少平底斜腹盖形碗；多小型浅腹平底盆，少双腹盆；少鼎、单耳杯

#### 夏家店下层（商时期）

- 辽西型
  - 分布：辽宁辽河以西、河北长城以北：**渤海湾北部**
  - **半地穴房**，**两层围墙（1石块；2土坯，个别永夯土，个别用泥土）**
  - 多作圆形袋状窖穴，也有圆形直筒窖穴（都来自河北龙山）
  - 石棺石椁
  - 陶范法铸造铜器（唐山大城山遗址红铜牌），晚期有大礼器（可能受到商影响）
  - 磨制石器、打制石器的半成品、细石器
  - 肩胛骨或**长方形骨片**（不见于其他文化），先钻后灼
  - 卷缘桶状鬲盛行，彩绘黑陶（源于河北龙山雪山型）
- 燕山型
  - 分布：长城以南：北京为中心
  - 半地穴房
  - *未提到窖穴内容*
  - **土穴墓葬**，极少见石棺石椁
  - 成套青铜礼器
  - 黄金臂钏、耳环（未见于商墓，是**北方特点**）
  - 少见卷缘筒状鬲和彩陶

#### 平谷刘家河近商

区别：

1. 器类“铜当卢”不见于商器
2. 三足卣为壶形，与敖汉旗大甸子黑陶壶相似，不近商卣
3. 铜甗甑部浅腹，与北票丰下陶甗相似
4. 盘内鱼纹不同于商器

#### “肃慎、燕亳，吾北土也”

- 肃慎/息慎：舜纳肃慎，渤海湾附近；商纣时孤竹伯夷（国名）居渤海之滨；汉“挹娄”（国名，但地点迁移到吉林黑龙江地区）
- 西周燕亳（邦）：易县“燕下都”（战国中期称法），西周燕始封地在琉璃河中都
  - “燕”字金文旧释“矣”，玄鸟形，燕图腾为鸟。商器用“燕”，西周初期出现“匽”，《亚盉》燕、匽二字同时出现（过渡阶段）

### 3.河北龙山许坦型

- 山西龙山文化分布
  - 东边：河北龙山涧沟型
  - 西边：陕西陇山
  - 南（晋西南、汾水下游）：**河南龙山晋西南型（夏文化东下冯型的来源）**
  - 北（晋中、晋西北）：河北龙山许坦型
- **光社文化**在太原附近，在晋东北与夏家店文化重叠（分布区域还不十分清楚）
- 特点
  - 第一类：许坦的单柄乱绳纹斝（单柄鬲即斝）、素直领绳纹鬲、单柄光面杯、敛口钵，东太堡带槽骨刀、单柄篮纹罐，光社白灰面、石刀、石环、陶环
    - 似河北龙山涧沟。单柄杯似河南龙山
    - 定名河北龙山许坦型
  - 第二类：光社的扁足平口蛋形瓮鼎、外表有细绳纹的高锥足根鬲足、扁平石铲、有灼无钻的猪肩胛骨卜骨、单柄素领细绳纹斝，东太堡素高领细绳纹鬲、卷领细绳纹盆，狄村筒流爵、扁足盆鼎、鬲、甗、斝、折腹盆、豆，许坦扁足盆形鼎、碗形豆
    - 平口蛋形瓮鼎、碗形豆 -> 夏文化东下冯型、先商漳河型
    - 单把鼎、<u>筒流绝</u> -> 二里头晚期
    - 楔形点纹、扁足盆形鼎、甗 -> 先商漳河型
    - 斝 -> 南关外型和早商
    - 单柄袋足鬲、高领鬲、筒流爵、折腹盆 -> **夏家店下层**
    - 夏文化晚期至早商
  - 第三类：光社联裆锥足或柱足、楔形纹、细绳纹鬲、堆纹甗腰、楔形纹、绳纹罐、有肩石铲、只钻不凿的卜骨，许坦石棺墓、陶甗、绳纹簋，东太堡陶甗、绳纹罐、楔形纹绳纹罐
    - 楔形纹鬲 -> 先商漳河型
    - 石棺墓、绳纹簋、浅腹甗、堆纹甗腰、绳纹罐 -> 夏家店下层
    - 粗绳纹矮足甗 -> 早商VIII组
  - 第四类：光社三角划纹鬶或罍片、方肩罐口
    - 三角划纹，年代相当于殷墟晚期

鼎、觚、爵、斝、卣、瓿。觚爵斝、觚爵组合

- 不见于殷墟
  - 豆（殷墟晚期陶豆形制）、龙纹觥
  - 长条两面刃匕，柄镂空作蛇头状，藁城有羊首状的，山西传入
  - 銎内刀：来源先周（偏北方的西周早期墓也有），源于晋陕之间
  - 铃首剑：短剑，北方游牧民族传入
  - 弓形器（保德林为金制），放在死者胸部，西周弓形器族徽，先周时在太原东寿阳出土
  - 车马器：与商器异制，夏家店上层相似
  - 金耳环（石楼后兰家沟、下辛角村）：夏家店下层，不见于商（有玦无耳环）

#### 鬼方、土方、𢀛方、燕京之戎

夏、商统治边界在晋西南汾水下游、涑水流域

- 虞：荤粥/熏粥/山戎
- 夏：淳维
- 殷：鬼方/鬼戎
- 周：玁（简体字用猃）狁二族，包括土方、𢀛方
- 汉：匈奴

##### 土方

武丁伐土方

1. 卜辞中土方在石楼，汉有“土”名，见唐《元和郡县图志》石楼县 -> 汉土军县，魏孝文帝置吐京郡
2. 安阳西稍偏**北**，符合卜辞方位（“⿰⿱一㞢又”在沁阳之北，有来㛸自北）
3. 石楼县应该包括在光社文化之内的考古学文化，墓葬和铜器，有武丁时代
4. “自土沮漆”，周人本居汾，古工去汾至陕，可以从石楼渡黄河
   - “光社文化应该包含在姬周文化之内”

𢀛方在其附近

1. 王国维释“𢀛”为“吉”，《元和郡县图志》吉昌县，汉北屈县
2. 有待考古学证据

### 4.共工氏

- 共工
  - 与颛顼争为帝
  - 尧之水官
    - “舜之时，共工振滔洪水，以薄空桑”（郝懿行：河南西部/莘虢之间、或山西北部/赵代间、或山东西部/兖地）。
  - 被尧舜流放幽州/幽洲/幽都/幽陵（北京昌平密云）
  - 遭禹之攻逐
  - **有子曰句龙，为后土。能平九土，故祀以为社**
- 徐旭生《中国古史的传说时代》考证：
  - 共水：在山西或豫西（《山海经》）
  - 共国：甘肃泾川灵台（《大雅·皇矣》）、或**河南辉县**-汉共县（《汉书·地理志》）

太行山东麓一带，所处文化应是河北龙山文化（各种类型都是，下据族徽分析——沿太行山东西两线自南向北都有共工族分布）

#### ![](ran.png)族

- “冉”，亦“篑”
  - 《广雅》：“簣<sub>苦怪</sub>、篣<sub>彭、</sub>笯<sub>女加奴慕、</sub>簝<sub>力麼</sub>、籝<sub>盈、</sub>篝<sub>溝、</sub>笭<sub>零</sub>，籠<sub>力公</sub>也”，“篝”也从“冉”
  - 篑：“未成一篑”，挑土用土笼
  - “冉”像簸箕形
  - “其”是“冉”倒置，都是土笼（篑）的象形
  - “爯，并举也。”（因而宋代金石书多释此族徽为“举”）
- 句龙平九土
  - 平水土用铲的方法
  - 夯筑法，汤阴白营河北龙山发现夯土
  - 徐旭生《中国古史的传说时代》：淇水初入平原，纳共水（洪水）奔腾冲击构成大患
- “益”，本作![](yi.png)
  - “冉”族徽头上加“廿”/共字头（伪变）
  - “益化”铜钱：山东益都，济南、青岛。山西苍底村“益”字圜钱。临淄“益”陶文
  - “共”字本身可能是“益”的误读
- 族徽分布
  - 大部分为商器，最早武丁晚期，有少量周器，最晚穆王（与河北龙山时代大体相当）
  - 邶器。邶未必是燕，但与燕接壤
  - 河北西南、西北、东北，河南北，辽宁西，山西大部（冀州之域）

“某种意义上说，这两者（共工氏、河北龙山）可能就是一回事。”
