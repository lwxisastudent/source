# 3.国际音标

### 1.音素：人耳可分辨的最小语音单位

### 2.历史

- 1886语音教师协会→1897国际语音学会
- 本来用于用语音学教英语
- 《语音教师》→《国际语音学报》

### 3.制定原则

一音一符，一符一音

### 4.形式

- 小写字母：印刷体、书写体、颠倒、加笔画（舌尖后/卷舌辅音用）、合体
- 其他语言字母
- 新造字母
- 附加符号[◌̃]鼻化，[◌̥]清音化，[◌ʲ]颚化，[◌ʰ]、[◌ʱ]

组合用符号复制（上波浪、下圆圈、上圆圈）：

```
̃
```

```
̥
```

```
̊
```

每个音标只有一种写法

### 5.易混

- ɤ是元音，ɣ是辅音

- ø元音，ɸ辅音，∅零声母

- ʮ舌尖元音，ɥ是半元音（近音）
- ɵ是元音，θ是辅音

### 6.元音和辅音

1. 气流不受阻碍
2. 声道各部位均匀保持紧张
3. 气流弱
4. 原因声带颤动，辅音不一定
5. 声学上，元音为周期波，复印多是非周期波

#### 1.元音第一课：基本（标准/正则）元音+ᴀ，表中红色

e：有点张不开，不自然，比普通话高（开口笑）

ɛ：有掉下巴的感觉

#### 2.元音第二课：次基本元音，表中蓝色

#### 3.元音第三课：表中绿色、粉色

ᴇ：自然的e

ɪ、ʏ、ʊ：松散的i、y、u（更松弛，嘴张大些）

#### 4.元音第四课：舌尖元音

**舌中线为马鞍形，舌面和舌尖都是舌高点**

|        | 不圆唇 | 圆唇 |
| ------ | ------ | ---- |
| 舌尖前 | ɿ      | ʮ    |
| 舌尖后 | ʅ      | ʯ    |

#### 5.元音第五课：卷舌元音

**发舌面元音时舌尖后缩并向硬腭翘起**，用[◌̴]或加[◌r]（也只是标记，单元音，不代表两个音。已经约定俗成，使用元音和辅音[r]组合时需要另加说明）

#### 6.元音第六课：鼻化（口鼻）元音（半鼻音）

上面加[◌̃]

汉语方言中似乎没有单独的鼻化元音

#### 7.元音第七课：复合元音

*/na.ˈiv/ ≠ /ˈnai̯v/（不成音节符号。汉语中复元音多，可简化）*

发音器官位置有变化（动程）：[ai]经历æ、ɛ、ᴇ、e等

有主次之分，一般只有一个原因响（假性复元音），如[ia]为后响，[ai]为前响，三合元音通常为中响。同时响亮的为真性复元音（存在存疑），厦门话中有“又”[iu]。

汉语方言中的特殊复元音：

1. 真性复元音
2. 包含长元音的复元音（复元音结合松散）<br>广东话街[ka:i]鸡[kɐi]、考[a:u]口[ɐu]（是对应不同韵母的规律情况）
3. 动程极短的复元音<br>（北京话[æi]动程长）陕西兴平话[mæɛ]动程短，接近单元音
4. 跳跃性过渡<br>“假前响”“假后响”：祁门话[i:ɐ] [u:ɐ]

### 8.声调

#### 8.1.五度标调法

声调本质上是相对音高，不同人绝对音高不同

- 声调符号：竖线左边分五点四段（Unicode 占位修饰符号）。短调《中国通用音标符号集》用窄声调符号，《方言》用下划线。
- 数字标音：右上角标单字调，右下角标变调。短调用下划线，5=<u>55</u>。

调型：

| 平调   | 11、33、55 |
| ------ | ---------- |
| 升调   | 13、35、15 |
| 降调   | 31、53、51 |
| 升降调 | 313、535   |
| 降升调 | 131、353   |

#### 8.2.声调和声调语言

超音质成分：音高、音强、音长

**音高变化为声调**

#### 8.3.调类

调类：一种方言中调值归类（按可独立区别）

##### （1）调名：

- 对外汉语教学中用：第一、二、三、四调
- 按中古四声关系定调：四声八调

###### a.发圈法

逆时针从左下起步，缺口都朝内，阳调加下划线：꜀◌꜁◌꜂◌꜃◌◌꜄◌꜅◌꜆◌꜇

###### b.标数字

##### （2）按声母是否送气再分调（后期形成）

（全：不送气，次：送气）

- 次清&次浊→次阴&次阳

### 9.音节（基本语音单位）划分

响度说和肌肉紧张说（西欧、休都是[ɕiou]，要看肌肉紧张位置）

为自己母语分音节往往自然统一。与语音社会性有关。golf，英语为一个音节，汉语母语者自然拆成三个。

汉语以字音为音节

### 10.汉语音节结构（传统音韵学与现代语音学融合）

1. 一般以元音（V）为核心，也有以辅音（C）为核心【一般为浊辅音，常见闭塞音 苏州话姆[m̩<sup>44</sup>] 唔[n̩<sup>44</sup>] 鱼[ŋ̍<sup>223</sup>] 面[l̩<sup>223</sup>]（只剩声母而成），陈老师漳平话中行[ŋ̍<sup>24</sup>]（只剩韵尾而成）】
2. 四种基本类型：V、CV（开音节），VC、CVC（闭音节）
3. 特点：
   1. 复元音丰富
      1. 三合原因一般只在开音节中（反例建瓯话返[xuaiŋ<sup>21</sup>]），首末元音一般不同（反例温州话吃[tɕ‘iai<sup>323</sup>]）
   2. 复辅音一般不允许（反例江西话食[st]，很少见！有单辅音演变hm hn hŋ
   3. 闭音节结尾的C一般为鼻音或塞音（反例江西都昌笔[pil]），一般没有双尾韵（闽北有，海南剪[tsienʔ]）
   4. 一个音节一般四个音素以内（海南剪）
4. 结构分析：

\begin{cases}

韵母：韵头、韵腹、韵尾\\\\
声调
\end{cases}
$$

#### 11.2.按发音方法

- 闭塞辅音：完全闭塞
  1. 塞音：通过一下子打开除阻（爆破音）
     - [pᴀ]、slee**p**都是除阻期发声，[lip]成阻期发生（只有闭合没有打开，“唯闭音”），为“暂音”，持阻期不能发声
  2. 鼻（塞）音：鼻腔开放
     - 发生时期有三种，[m]可一直发声
     - 一般为浊音，清化鼻音（根本不带声音，擤鼻涕声）也出现，如贵州威宁苗语炒“面”[m̥au]
  3. 塞擦音：先塞后擦，塞音除阻+擦音持阻，发塞音时迅速打开一条缝，可视为塞音的持阻发声
     - IPA表没有单独的塞擦音符号，用两个辅音加连音符t͡s或合字ʦ；由于汉语方言常用，可直接使用，不视为复辅音
- 间隙辅音
  1. 擦音：中缝擦音（中间有条缝）[f] [s] [x]
     - 通音：也是中缝，但缝较宽，几乎无摩擦。唇齿[ʋ]（北京话“未”声母）；red、汉语拼音r [ɻ]；半元音[j] [w] [ɥ]（北京话遥[i<sub>j</sub>au]、王[u<sub>w</sub>ɑŋ]、云[y<sub>ɥ</sub>n]
  2. 边音：中间堵塞，两边留缝
     - 一般为浊音
     - 除阻发声：汉语拼音l。持阻发声苏州话面[l̩<sup>223</sup>]
  3. 颤闪音：音色易变，不同人不一样
     - 颤音：连续不强烈爆发音的组合。双唇[ʙ]（打嘟噜，西安回民[t<sup>ʙ</sup>u]），舌尖[r]（赶驴马trrr），小舌[ʀ]
     - 闪音：稍微碰一下立刻收回。日语中r（五十音图ら行ら[ra]り[ri]る[ru]れ[re]ろ[ro]）是舌尖中闪音[ɾ]；舌尖后闪音[ɽ]
